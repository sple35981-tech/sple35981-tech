# Noxen Profile Redesign

## Goal
Replace the cyberpunk identity page with a restrained personal index that feels edited by one person rather than generated from a GitHub profile template.

## Visual direction
- Editorial monochrome layout with one muted red accent.
- Static cover artwork based on debugger addresses and a subtle radar construction.
- No gradients, glow, particles, planets, fake terminals, binary text, typing animations, badges, counters or skill bars.
- Typography and spacing carry the design.

## README
- No biography section.
- One static cover, three selected destinations and a three-line working rule.
- Keep the page under 60 source lines and usable in both GitHub themes.

## GitHub Pages
- Replace Three.js with a dependency-free HTML/CSS/JavaScript index.
- Present projects as an editorial work list with an address rail and desk notes.
- Support J/K and arrow-key navigation, Enter to open, reduced motion and responsive layouts.

## Success criteria
- The previous cyber/AI phrases and effects are absent.
- All local assets resolve.
- SVG and HTML parse successfully.
- The site has no external script dependency.
